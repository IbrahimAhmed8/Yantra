01 — VISION & MISSION
Why Yantra Exists
The global skills gap in AI and computer science is not a content problem — it is a guidance problem. Millions of
students have access to YouTube, Wikipedia, and free courses, yet most quit within weeks because nobody is
guiding them. Yantra fixes this by deploying an AI teacher that knows exactly where each student is, where they
need to go, and how to get them there — while making the journey hands-on, interactive, and genuinely fun.

04 — PRACTICE ROOMS (VOIDS)

about neural networks, students build one. Instead of watching a video about gradient descent, they drag a slider
and watch the loss curve change in real time. The AI teacher is present inside every Void, watching what the
student does, catching mistakes before they compound, and celebrating wins.

05 — AI TEACHER PERSONA
The Personality Engine
The AI teacher is not a search engine with a chat interface. It has a persistent personality, a teaching
philosophy, and adaptive communication style. The persona is implemented as a carefully crafted system
prompt combined with a dynamic context injection layer that loads the student's profile, current topic, and
emotional state before every response.
Core Personality Traits
Trait
Behaviour
Example
Patient
Never rushes, always willing to re-explain differently
"Let me try explaining this a different way..."
Encouraging
Celebrates small wins, never makes student feel dumb
"That's a great first attempt — here's one tweak:"
Socratic
Asks questions back to build understanding, not just answers
"What do you think happens if we change the learning rate?"
Contextual
Knows what the student learned last session
"Last time you struggled with activation functions — today we'll fix that"
Honest
Says when it doesn't know, corrects mistakes clearly
"Actually that's a common misconception — let me clarify"
Age-aware
Speaks differently to a 13-year-old vs a 25-year-old
Adjusts vocabulary, examples, complexity automatically
Pedagogical Framework
Yantra's AI teaching approach is grounded in established learning science: Bloom's Taxonomy (remember →
understand → apply → analyse → evaluate → create), spaced repetition for long-term retention, active recall
over passive reading, and worked examples before independent practice.
Learning Science Principle
How Yantra Implements It
Spaced repetition
AI re-introduces topics at optimal intervals based on forgetting curve model
Active recall
Quiz Void triggered after every major concept, not just at end of module
Worked examples → fading
AI shows full solution first, then provides partial scaffolding, then blank
Interleaving
Practice mixes multiple topics to strengthen connections between concepts
Growth mindset language
System prompt explicitly avoids fixed-mindset phrases, celebrates effort not talent
Zone of proximal development
Skill engine keeps challenges 15–25% above current ability (Vygotsky sweet spot)

when a diagram would help more than text, when a quiz is appropriate, or when live code would make the concept
click.

Trigger

07 — SMARTBOARD INTEGRATION
Classroom-First Design
Smartboards in Indian and global schools run Android, Windows, or Linux — and all of them have a full browser.
Yantra's Classroom Mode is a fullscreen web app that works out of the box on any smartboard without installing
anything. The school just opens a URL. The teacher logs in, selects today's topic, and the AI takes over.
Feature
Description
Kiosk mode URL
Single URL loads fullscreen classroom UI, no distractions, touch-optimised
Teacher dashboard
Teacher sees all student screens simultaneously, can spotlight one student's Void
Live class mode
AI teacher presents concept to whole class on big screen, students follow on tablets
Student sync
Students' individual sessions sync with the classroom board — teacher can 'pull' any student's work
Offline mode
Core content cached via Service Worker — works even without stable internet
Voice interaction
Web Speech API enables voice questions to the AI teacher (great for smartboard use)
Touch gestures
All Voids fully touch-optimised — pinch to zoom code, tap to run, swipe to next challenge
Class analytics
Teacher sees real-time class progress, who's stuck, who's ahead, topic heatmap
Embedding Yantra in Other School Software
For schools using an LMS (Learning Management System like Moodle, Google Classroom, or custom software),
Yantra provides an iframe embed SDK and a REST API so any platform can add an AI teacher with 10 lines of
code:
data-topic="neural-networks"
data-student-id="{{student_id}}"
data-mode="classroom">
<br/> Yantra.init({ apiKey: "YOUR_KEY", theme: "dark" });<br/>
